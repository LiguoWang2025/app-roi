## [2026-03-20] 上传流程优化

### Issue
CSV 上传完成后需要手动关闭对话框并手动刷新数据，用户体验不够流畅。

### Cause
原实现缺少：
- 上传成功后自动关闭对话框
- 上传成功后自动刷新数据
- 友好的成功提示信息

### Solution
优化了上传流程，实现自动化操作：

**1. UploadModal 组件更新 (`components/UploadModal.tsx`)：**
- 添加 `onUploadComplete` 可选回调属性
- 上传成功后自动调用回调并关闭对话框
- 保持点击遮罩层关闭功能

**2. CsvUploader 组件更新 (`components/CsvUploader.tsx`)：**
- 添加 `onUploadComplete` 回调支持
- 上传成功后延迟 1.5 秒自动触发回调
- 延迟时间让用户看到成功提示
- 添加"正在刷新数据..."提示
- 优化暗色模式样式支持

**3. 主页面更新 (`app/page.tsx`)：**
- 从 `useROI` hook 获取 `refetch` 函数
- 创建 `handleUploadComplete` 处理函数
- 上传完成后自动刷新 ROI 数据
- 传递回调到 UploadModal 组件

**工作流程：**
```
1. 用户上传 CSV 文件
2. 后端处理数据
3. 显示成功提示（1.5 秒）
4. 自动关闭对话框
5. 自动刷新图表数据
```

**用户体验改进：**
- ✓ 无需手动关闭对话框
- ✓ 无需手动刷新数据
- ✓ 清晰的成功反馈
- ✓ 流畅的视觉过渡
- ✓ 暗色模式完美支持

**样式优化：**
- 上传中：蓝色提示框
- 错误：红色提示框
- 成功：绿色提示框 + 刷新提示
- 所有状态支持暗色模式
